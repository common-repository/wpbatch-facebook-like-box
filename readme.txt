== Installation ==



1. Upload `wpbatch-facebook-like-box.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to `Appearance > Widgets` and you will get a new widget named WPBatch `Facebook Like Box`
4. Just Add the widget in any widget area, add page link, height and width.. and you are done.. Enjoy!!!

Good Luck
